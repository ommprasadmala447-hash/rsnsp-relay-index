Formula Documentation: ∆RSNSP Protection Scheme

This documentation explains the important formulas used in the Incremental Negative Sequence Power based protection scheme for AC microgrid feeders. Read More on IEEE Xplore

1. Negative Sequence Component Formula

Formula

ψx2 = 1/3 ( ψax + α2ψbx + αψcx )

Meaning

This formula is used to calculate the negative sequence voltage or negative sequence current at a bus.

Symbols

  • ψ = Voltage V or Current I
  • x = Bus end, either m or n
  • ψax, ψbx, ψcx = Phase a, b, c values
  • α = 1∠120°
  • ψx2 = Negative sequence component

2. Incremental Component Formula

Formula

Δψ(i)x2 = ψ(i)x2 - ψ(i-N)x2

Meaning

This formula calculates the change in negative sequence voltage or current after a fault occurs.

Symbols

  • Δψ(i)x2 = Incremental negative sequence value
  • ψ(i)x2 = Present sample value
  • ψ(i-N)x2 = Previous cycle sample value
  • N = Number of samples in one cycle

3. Incremental Negative Sequence Apparent Power

Formula

ΔSm2 = ΔVm2 × ΔIm2*

ΔSn2 = ΔVn2 × ΔIn2*

Meaning

These formulas calculate the incremental negative sequence apparent power at bus-m and bus-n.

Symbols

  • ΔSm2 = Incremental apparent power at bus-m
  • ΔSn2 = Incremental apparent power at bus-n
  • ΔVm2, ΔVn2 = Incremental negative sequence voltages
  • ΔIm2, ΔIn2 = Incremental negative sequence currents
  • * = Complex conjugate

4. Incremental Negative Sequence Real Power

Formula

ΔPm2 = Re(ΔSm2)

ΔPn2 = Re(ΔSn2)

Meaning

These formulas extract the real power part from the apparent power at both ends of the protected feeder.

5. Proposed Relaying Feature: ΔRSNSP

Formula

ΔRSNSP = ( ΔPm2 + ΔPn2 ) / ΔPmin

Calculate

Meaning

ΔRSNSP is the main fault detection index. It is calculated using the sum of incremental negative sequence real powers at both ends of the protected feeder.

Symbols

  • ΔRSNSP = Ratio of Sum of Negative Sequence Power
  • ΔPm2 = Incremental real power at bus-m
  • ΔPn2 = Incremental real power at bus-n
  • ΔPmin = Minimum value between the two powers

6. Minimum Power Selection Formula

Formula

If |ΔPm2| < |ΔPn2|, then ΔPmin = ΔPm2

If |ΔPn2| < |ΔPm2|, then ΔPmin = ΔPn2

If |ΔPm2| = |ΔPn2|, then ΔPmin = ΔPm2 or ΔPn2

7. Internal Fault Condition

Formula

ΔRSNSP ≥ 2

Meaning

If the fault is inside the protected feeder zone, the value of ΔRSNSP becomes greater than or equal to 2.

8. External Fault Condition

Formula

ΔRSNSP < 0

Meaning

If the fault is outside the protected zone, the value of ΔRSNSP becomes negative.

9. Threshold Formula

Formula

K = 1

Decision Rule

If ΔRSNSP > K, then internal fault is detected.

If ΔRSNSP ≤ K, then internal fault is not detected.

10. Protection Logic

  1. Measure three-phase voltage and current.
  2. Calculate negative sequence voltage and current.
  3. Calculate incremental components.
  4. Calculate incremental apparent power.
  5. Calculate incremental real power.
  6. Find ΔRSNSP.
  7. Compare ΔRSNSP with threshold K.
  8. If ΔRSNSP > 1, generate trip signal.

⚡ PAGE 2: FAULT TYPE CATEGORIES

Explanation of all fault types and how ΔRSNSP responds to each of them.

🟢 Single Line-to-Ground (SLG)

Button Meaning
A-G Phase A contacts ground
B-G Phase B contacts ground
C-G Phase C contacts ground
  • Only one phase is disturbed
  • Produces a small negative sequence component
  • Most common fault in real systems (~70%)

🔴 Double Line-to-Ground (DLG)

Button Meaning
AB-G Phase A and B contact ground
BC-G Phase B and C contact ground
CA-G Phase C and A contact ground
  • Two phases are disturbed simultaneously
  • Produces stronger negative sequence component than SLG

🟣 Three Phase-to-Ground

Button Meaning
ABC-G All three phases contact ground
ABC All three phases short together (no ground)
  • All three phases are disturbed equally
  • ABC-G gave 3.835 >> K=1 → TRIP
  • Considered the most severe fault type

🔵 No Fault

Button Meaning
NONE No fault applied
  • System runs in balanced condition
  • ΔRSNSP stays near 0
  • No trip signal generated

⭐ Key Mathematical Rule

  • Internal fault → both ΔPm2 and ΔPn2 are negative → their sum divided by minimum always gives ≥ 2
  • No fault/external → powers don't change significantly → ΔRSNSP stays below 1 or becomes negative

That's why your ABC-G test gave 3.835 — all three phases were disturbed, creating large negative sequence power.
